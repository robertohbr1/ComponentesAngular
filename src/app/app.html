Componente: Chips Selector

Um seletor reativo múltiplo que gera strings formatadas em padrão SQL (ex: 'A','B'), ideal para cláusulas IN de banco de dados.

Exemplo 1

Filtro por Letra Inicial

Valor de Retorno (Variável):
{{ selectedLetters() || '"" (Em branco - Todos selecionados)' }}
Aplicação em Consulta SQL: 
SELECT * FROM Clientes
WHERE Nome LIKE 
@if (selectedLetters()) {
  ANY (ARRAY[{{ selectedLetters() }}])
} @else {
  '%' -- Filtro inativo (Todos)
}
Exemplo 2

Filtro por Categoria

Valor de Retorno (Variável):
{{ selectedCategories() || '"" (Em branco - Todos selecionados)' }}
Aplicação em Consulta SQL: 
SELECT * FROM Produtos
@if (selectedCategories()) {
  WHERE Categoria IN ({{ selectedCategories() }})
} @else {
  -- Sem WHERE (Todos os produtos)
}

Componente: Seletor de Período de Datas

Um gerenciador de faixa de datas (Data Inicial e Data Final) com validação de período e suporte a seleção por dia/mês/ano ou apenas mês/ano.

Calendário

Modo: Dia / Mês / Ano

Início:
{{ startDateDay() || '""' }}
Fim:
{{ endDateDay() || '""' }}
Consulta SQL: 
SELECT * FROM Vendas
@if (startDateDay() && endDateDay() && isPeriodDayValid()) {
  WHERE DataVenda BETWEEN '{{ startDateDay() }}' AND '{{ endDateDay() }}'
} @else {
  -- Filtro incompleto ou inválido
}
Validade do Período: {{ isPeriodDayValid() ? 'Válido' : 'Inválido' }}
Competência

Modo: Mês / Ano

Início:
{{ startDateMonth() || '""' }}
Fim:
{{ endDateMonth() || '""' }}
Consulta SQL: 
SELECT * FROM Balancos
@if (startDateMonth() && endDateMonth() && isPeriodMonthValid()) {
  WHERE Competencia BETWEEN '{{ startDateMonth() }}' AND '{{ endDateMonth() }}'
} @else {
  -- Filtro incompleto ou inválido
}
Validade do Período: {{ isPeriodMonthValid() ? 'Válido' : 'Inválido' }}
